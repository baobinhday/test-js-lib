Cách sử dụng Button Library

Ví dụ 1: Sử dụng cơ bản

Code:

// Tạo button với nội dung cơ bản
const button = ButtonLib.createButton({
  buttonText: 'Nhấn để hiển thị nội dung',
  containerId: 'example1-container'
});

// Set nội dung cố định
button.setContent(`
  <div>
    <h3>Xin chào từ Button Library!</h3>
    <p>Đây là một ví dụ đơn giản về thư viện button.</p>
  </div>
`);

Ví dụ 2: Nội dung tùy chỉnh

Code:

// Tạo button với nội dung tùy chỉnh
const button = ButtonLib.createButton({
  buttonText: 'Hiển thị/Ẩn nội dung',
  containerId: 'example2-container'
});

// Set nội dung ban đầu
const textarea = document.getElementById('custom-content');
button.setContent(textarea.value);

// Cập nhật nội dung khi người dùng nhấn nút Update
document.getElementById('update-content').addEventListener('click', () => {
  button.setContent(textarea.value);
  
  // Hiển thị nội dung nếu đang ẩn
  if (!button.isContentVisible) {
    button.toggleContent();
  }
});
